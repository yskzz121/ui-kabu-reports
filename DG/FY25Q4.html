ビジネスモデル

米国最大の店舗数を誇るディスカウントリテーラー。EDLP（Everyday Low Price）戦略で、大半の商品を10ドル以下で提供。人口2万人以下の農村部・小規模コミュニティに注力する「スモールボックス」業態で、大型店の進出しにくいエリアに事実上の地域独占を構築。

収益の~82%が消耗品（食品・飲料・日用品）で高頻度来店を生み、PB（プライベートブランド）のClover Valley（年間$2.3B超）を軸に粗利率を改善中。新収益源のDG Media Network（リテールメディア）は年間~$170M規模に成長。

⚠️ リスク要因

  • 📌 関税影響: Q4でLIFO $45M影響。コスト40-50%が消費者に転嫁される見込み
  • 📌 コア顧客の購買力低下: 低所得層の財務状況悪化が継続。SNAP就労要件強化の影響リスク
  • 📌 ガソリン価格上昇: コア顧客の可処分所得を直接圧迫
  • 📌 FY2026 SSS成長率鈍化: ガイダンス+2.2-2.7%はFY2025の+3.0%から減速
  • 📌 WOTCの失効: FY2026のEPSに約-$0.13の影響
  • 📌 Walmartの低価格帯侵食: 食料品シェア72%到達、DGの顧客層と重複拡大
  • 📌 最低賃金引上げ: SGA費用の構造的な上昇圧力
  • 📌 粗利改善ペースの鈍化: CFOが「FY2025比で大幅に小さい」と明言

🚀 カタリスト

  • ✅ シュリンク削減の継続: さらに+50bpの粗利改善余地が残存
  • ✅ セルフレジ撤去の成功: 損失防止と顧客体験の両面で効果実証済み
  • ✅ DG Media Network拡大: $170M規模の高マージン広告収入が急成長中
  • ✅ PB商品（Clover Valley）の拡充: 年間$2.3B超の高マージン売上
  • ✅ 店舗マネージャー離職率-375bp: 売上・執行力改善の先行指標
  • ✅ Value Valley（$1商品）の成功: +17.6%コンプ成長
  • ✅ AI駆動SGA効率化: 長期目標に未織り込みの追加アップサイド
  • ✅ 景気後退耐性: 不況時にトレードダウン需要が増加するカウンターシクリカル特性
  • ✅ 機関投資家の買い増し: BlackRock+2.96%、State Street+9.09%
