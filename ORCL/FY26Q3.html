Oracleは1977年創業の世界最大級のエンタープライズソフトウェア企業。リレーショナルデータベース管理システム（RDBMS）分野での数十年に及ぶ支配的地位を基盤に、クラウドERP（Fusion Cloud / NetSuite）・自律型データベース・OCI（Oracle Cloud Infrastructure）へと事業をシフト中。法人向けDB市場での深い顧客関係と移行コストを武器に、AI時代のインフラ企業として急成長しており、特にOCI IaaSは直近で前年比70%超の成長を記録している。

事業モデル · Business Model

プロダクトライン

  • Cloud Services & License Support（~77%） — OCI（IaaS）+ Fusion Cloud ERP/HCM/SCM + NetSuite（SMB ERP）+ Autonomous Database。クラウド移行加速でサブスク比率が急上昇し主力収益源に
  • Cloud License & On-Premises License（~10%） — Oracle Database（オンプレ）ライセンス販売。既存大口顧客のOCI移行前橋渡し。BYOL（Bring Your Own License）でクラウドへの継続性を担保
  • Hardware / Services（~13%） — Oracle Exadata（データベース専用エンジニアドシステム）、コンサルティング・導入支援サービス。エンタープライズの基幹系システム基盤として継続的な需要

収益モデル

  • サブスクリプション（クラウド中心） — クラウド移行に伴いサブスク比率約77%超に到達。ADB（自律型DB）・Fusion ERP・NetSuiteが複数年契約を牽引しNRRが安定
  • ライセンス更新・サポート収益 — 既存オンプレ顧客の年次サポート費用。解約率が極めて低く安定した反復収益として機能。クラウド移行促進のクロスセル起点
  • OCI IaaS成長（急拡大） — OCI IaaS売上がYoY+84%（FY26Q3）で急成長。GPUクラスタ需要取り込みによりFY27通年売上 ~$90Bのガイダンスを発表

Competitive Moat · 競争優位性

  • データベース依存の超高スイッチングコスト — 大企業の基幹系DB・ERPがOracle前提で構築されており、移行コスト・リスクが極大。Fortune 500の多くがOracle DBを40年以上継続利用し、競合への切替は事実上の多年度プロジェクトを要する
  • マルチクラウド展開による生態系防衛 — Oracle Database@Azure・Oracle@AWSにより、顧客がAzureやAWSを選んでもOracle DBを利用継続できる構造を構築。競合インフラを「流通チャネル」に転換する巧みな戦略
  • Autonomous Database（自律型DB）の技術的先行 — AIによる自動パッチ・チューニング・スケーリングを実現したADBは競合にない独自機能。DBaaS市場でのコスト優位と運用工数削減が既存顧客の移行先としてOCIを優位に立たせる
  • Fusion ERP + OCI の水平統合による粘着性 — クラウドERP（Fusion / NetSuite）とOCIインフラを一体提供することで、顧客の業務データ・ワークフローがOracle基盤に深く統合。単独製品よりも離脱コストが大幅に上昇
  • AIデータセンター需要への乗り換え — Stargate（OpenAI）などAI超大規模プロジェクトのインフラ受託により、既存DBビジネスとは別のAI IaaS成長軌道を獲得。100超のクラウドリージョン展開速度が競合優位
