今回のポイント
Non-GAAP EPS +$0.10 — 上場来初の黒字転換が示す収益モデルの転換点
コンセンサスが -$0.17 を予想する中、実績は +$0.10 と $0.27 の大幅ビートを達成。SBC負担の大きいGAAP指標では依然赤字だが、実質的なキャッシュ収益力の獲得を意味する。この転換は投資家の間で長らく待たれていたマイルストーンであり、株価の再評価余地が開く。
FCF $76.9M (+394% YoY) — 約5倍の急拡大でキャッシュジェネレーターへ
FCFマージン約22%は成熟SaaS企業並みの水準。前年 $15.6M からの急拡大は、顧客基盤のスケールと前払いサブスクリプション収益の積み上がりが背景。現金 $1.6B を保有しつつ $1.1B の転換社債も残るが、FCFの急増により財務的な懸念は後退している。
Cloud ARR 87%（+53% YoY） — クラウドシフト加速で高成長の持続性を確認
全ARR $1.35B のうち Cloud ARR は $1.17B（前年比 +53%）を占め、87%のクラウド比率を達成。オンプレミスからクラウドへの移行が加速しており、サブスクリプション型収益の比率向上がRevenue Qualityを高めている。Net New Subscription ARR $94M は過去最高更新。
Rubrik Agent Cloud — 独立製品スイートとして展開するAI戦略の本格化
CEOはRubrik Agent Cloudを「完全に独立した製品スイート」と位置づけ、AI変革需要への対応を表明。Okta Recoveryと CrowdStrikeとの連携により、IDセキュリティ領域への拡張が進む。CISOへの浸透は「わずか3四半期で顕著な初期トラクション」と経営陣が言及した。
Q4ガイダンス +30% — 成長鈍化を受け入れながら収益性を優先するシフト
Q4 FY26ガイダンスは $363〜$365M（YoY +30%）とコンセンサス $358M を上回ったが、Q3の正規化後 +36% からの鈍化は明確。成長率低下はCohesity+Veritas統合（$7B）・Veeam AI買収など競合激化の影響もあるが、経営陣は「成長から収益性へのシフト」を意図的に選択している局面とも読み取れる。
