🏛️ 企業概要・ビジネスモデル

基本情報

  • 設立: 2014年 / IPO: 2024年4月（$32/株）
  • 本社: カリフォルニア州パロアルト
  • CEO: Bipul Sinha（共同創業者）
  • 従業員: 約3,200人（FY26末）→ 約4,900人（2026年2月推計）
  • 事業: Zero Trust Data Security プラットフォーム
  • 時価総額: ~$10.43B

ビジネスモデル・収益構造

  • サブスクリプション比率: 96.6%（SaaS中心）
  • イミュータブルファイルシステム + 論理的エアギャップ
  • CDM（オンプレ）→ RSC（SaaS）移行推進中
  • NRR 120%超のランド＆エクスパンドモデル

主要製品・サービス

  • Rubrik Security Cloud (RSC) — 統合セキュリティプラットフォーム
  • Annapurna — AI搭載データセキュリティ
  • Agent Cloud — 新GA（AIエージェントセキュリティ）
  • Identity Recovery — 900社突破（史上最速成長）

競合環境

  • 主要競合: Cohesity（Veritas統合）、Veeam、Commvault、Dell EMC
  • Gartner MQ: リーダー象限（ビジョン完全性でトップ）
  • TAM: $363B（2024末）→ $529B（2027末）
  • 市場シェア: 7位（成長途上）

⚠️ リスク・カタリスト

⚠️ リスク要因

  • ARR成長率の減速（38%→34%、FY27ガイダンス25-26%へさらに鈍化）
  • 高バリュエーション（PSR 7.9x TTM）と株価下落リスク
  • インサイダー大量売却（直近90日で-$1,160万、買い0件）
  • 空売り比率9.5%と増加傾向
  • Cohesity-Veritas統合による競合激化
  • GAAP黒字化は依然未達（SBC $329M）
  • ソフトウェアセクター全体の弱い地合い

🚀 カタリスト

  • FY27黒字転換ガイダンス（Non-GAAP EPS $0.07-$0.27）
  • Identity製品の急成長（900社、史上最速プロダクト）
  • Agent Cloud GA — AIセキュリティの新市場開拓
  • 規制追い風（DORA、NIS2、SEC）でサイバーレジリエンス需要構造的増加
  • ランサムウェア脅威の継続的増加（+34% YoY）
  • FCF $237.8M（FCFマージン18%）で自力成長可能
  • アナリスト平均目標$87-99（現在株価の+72-95%上方）
