🏢 企業概要

基本情報

  • 設立: 2014年（Palo Alto, CA）
  • CEO: Bipul Sinha（共同創業者）
  • 従業員: ~3,500名
  • 上場: 2024年4月 NYSE @ $32/share
  • Board: John W. Thompson（元Microsoft会長）
  • 事業: Zero Trust Data Security
  • 収益モデル: 100%サブスクリプション

主要製品・サービス

  • Rubrik Security Cloud: データ保護・脅威分析・DSPM
  • Ruby: AI搭載ブリーチ調査アシスタント
  • Rubrik Agent Cloud: AIエージェントのガバナンス・可視化（FY26 Q4 GA）
  • Identity Resilience: ID保護プラットフォーム（FY26 Q3発売）
  • Atlas File System: イミュータブルデータ保護

競合環境

  • Cohesity: Veritas統合で最大の競合に（バックアップ統合）
  • Commvault (CVLT): エンタープライズバックアップの老舗
  • Dell Technologies: 自社ハードウェアとの統合が強み
  • 競争優位: Zero Trust Architectureによるセキュリティ差別化

市場機会

  • TAM: サイバーレジリエンス市場 ~$400B+
  • トレンド: AIセキュリティ、ランサムウェア対策需要の急増
  • 規制追い風: DORA, NIS2, FedRAMPなど
  • FY26テーマ: 「サイバーセキュリティからAIセキュリティプラットフォームへの進化」

🏆 FY2026 年間ハイライト・マイルストーン
Q1 FY26（2025年5月発表）
FY2026ガイダンス上方修正。FCF黒字転換を達成し、ARR $1.18Bで38%成長を記録。サイバーレジリエンス需要の強さを確認。
Q2 FY26（2025年9月発表）
売上成長率51%に加速（FY2026四半期最高値）。Non-GAAP EPSが-$0.03まで改善し、黒字化が間近に。サブスクリプションモデルへの完全移行が加速。
Q3 FY26（2025年12月発表）
Non-GAAP初黒字（EPS $0.10）達成。Identity Resilience製品発売で65件成約（40%が新規顧客）。Net New ARR $94M達成（当時の記録）。
Q4 FY26（2026年3月発表）
Net New ARR $115M記録更新。Rubrik Agent Cloud GA発表（Amazon Bedrock AgentCore・Microsoft Copilot Studioと統合）。ARR $1.46B達成。FY2027で完全黒字化の見通し。

⚠️ リスク要因

  • ARR成長率の減速傾向（38%→36%→34%→34%→ガイダンス26%）
  • GAAP大幅赤字継続（-$345M営業損失、-$349M純損失）
  • SBCが売上の25%（$329M）で依然高水準
  • 高バリュエーション（PSR 8x+）で金利感応度が高い
  • Cohesity-Veritas統合による競合激化
  • 株価がIPO後高値$103から45%下落
  • Material rights収益減少によるFY2027報告成長率ヘッドウインド
  • Short interest 7.1%で空売り圧力あり

  • Rubrik Agent Cloud GA → AIセキュリティ新市場への参入

  • Identity Resilience成長 → TAM拡大
  • NRR >120%持続 → セキュリティ製品クロスセル拡大
