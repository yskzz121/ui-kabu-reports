Rubrikはデータセキュリティ・サイバーレジリエンスに特化したクラウドネイティブ企業。ランサムウェア攻撃・内部脅威・データ破損に対し、バックアップ・リカバリ・脅威検知・データ可観測性を統合プラットフォーム「Rubrik Security Cloud」で提供する。クラウド・オンプレ・SaaSを問わず企業データを保護し、サイバー攻撃後の迅速な復旧（サイバーリカバリー）を武器に従来型バックアップベンダーからの置き換えを加速している。

事業モデル · Business Model

プロダクトライン

  • Rubrik Security Cloud（コアプラットフォーム） — データ保護・脅威アナリティクス・ランサムウェア検知・データポスチャー管理を統合提供。SaaS型サブスクリプションで課金
  • Ransomware Investigation & Recovery — 感染範囲の特定・クリーンポイントへの自動ロールバック・フォレンジクス支援。CISO・インシデントレスポンスチーム向けの高付加価値機能
  • Cloud & SaaS Protection — AWS・Azure・GCP上のワークロード、Microsoft 365・Salesforceなどのクラウドアプリケーションデータを対象とした保護と可視化

収益モデル

  • サブスクリプション主体（~90%超） — Rubrik Security CloudのARRベース課金。NRR120%超を維持しながら既存顧客からのアップセルが成長を牽引
  • アプライアンス（漸減） — ハードウェア一体型のオンプレ製品。クラウド移行に伴いサブスクへの移行が進行中
  • プロフェッショナルサービス — 導入支援・トレーニング・インシデント対応支援。本体サブスク契約に付随する補完的収益源

Competitive Moat · 競争優位性

  • セキュリティとバックアップの統合プラットフォームによる切替コスト — データ保護インフラはミッションクリティカルで、複数年かけて構築されたポリシー・ワークフローの移行コストが極めて高い
  • 不変バックアップアーキテクチャ（Zero Trust Data Security） — 攻撃者が侵入してもバックアップデータを改ざん・削除できない設計。競合が後付けで模倣困難な構造的優位
  • AIを活用したデータ脅威アナリティクス — 大量のバックアップデータを学習したAIがランサムウェアの異常を早期検知。データ資産が増えるほど検知精度が上がる正のフィードバックループ
  • Microsoft / AWS / Googleとの戦略的パートナーシップ — 主要クラウドベンダーとの深い統合により、クラウドネイティブ移行の文脈で優先的に選定される流通優位
  • 規制環境の追い風（NIS2・DORA等） — EU圏でのサイバーレジリエンス規制強化が義務的導入需要を創出。規制準拠のユースケースとして選定されやすいポジションを確立

今回のポイント · Key Insights
1. 売上高 $278.5MYoY +49%— コンセンサス $18.5M 超過
サブスクリプション売上は $265.7M（+54% YoY）で全体の95.4%を占め、継続課金モデルの強固さを示した。Americas が $203M（+51% YoY）と牽引役を担い、国際地域も $75M（+43% YoY）と加速。ランサムウェア対策需要の構造的拡大が背景にある。
2. FCF +$33.3M黒字転換— 前年比 $70M 改善
営業CFも +$39.7M と前年 -$31.4M から黒字化。現金残高 $762.1M と手元流動性は潤沢で、成長投資と財務安定の両立が進む。Sub ARR Contribution Margin の +8.0% 転換と併せ、収益構造改善が数値に明確に反映された。
3. Subscription ARR $1.18BYoY +38%— Cloud ARR は +60%
Cloud ARR は $972M（全ARRの82%）と急速にクラウドシフトが進む。NRR 120%超を維持し、既存顧客拡張のランドアンドエクスパンド戦略が機能。$100K+顧客は 2,381 社（+28% YoY）に拡大し、大企業への浸透が続く。
4. FY2026 通期ガイダンス上方修正 — 売上 $1.179 – $1.189B
前四半期から再度の上方修正。ARR も $1.380 – $1.388B（+26 – 27%）へ引き上げ。Q2 ガイダンスも売上 $291 – $294M とコンセンサスを上回り、経営陣の自信の高さを示す。Identity Recovery 製品の採用加速が上振れ要因として言及された。
5. アナリスト 19/19名 が Buy 以上 — 目標株価中央値 $111.50
全19名のカバレッジアナリストが Buy/Strong Buy を維持。目標株価レンジ $97 – $130。IPO安値 $28.34 から約3.6倍に株価が上昇しており、高成長SaaSとしての市場評価が定着しつつある。ただし PSR 13 – 16x 程度と高バリュエーションにある点は注意が必要。

Risk & Catalyst · リスク・注目ポイント / カタリスト

Positive Catalysts

  • Cloud ARR YoY +73% — クラウドネイティブ移行需要の加速が持続
  • サイバーレジリエンス需要拡大 — ランサムウェア対策への企業投資増加
  • Microsoft 365バックアップ普及 — SaaS保護ニーズでARR拡張余地大
  • AIネイティブデータセキュリティ製品（Ruby AI）の商用化加速
  • 大企業大規模採用 — $1M+ディール増加でARPU上昇
  • 国際展開加速 — EMEA・APAC での新規顧客獲得

Risk Factors

  • 競合激化 — Cohesity・Veeam・Commvault との価格競争・機能競争
  • 高評価による株価ボラティリティ — PSR 22x超で市場期待値が高い
  • 営業赤字継続 — Non-GAAP営業損失改善中だが黒字化は先
  • 訴訟リスク — Cohesity特許係争の帰趨次第でコスト増の可能性
  • 大型案件サイクル長期化 — エンタープライズ販売のパイプライン変動
  • マクロIT支出減速 — 景気後退局面での予算凍結リスク
