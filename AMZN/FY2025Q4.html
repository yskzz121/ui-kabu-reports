🏢 企業概要

Amazon.com（AMZN）は1994年ジェフ・ベゾスにより創業、ワシントン州シアトルに本社を置く世界最大級のテクノロジー・Eコマース企業。オンライン小売（北米・国際）、AWS（クラウドコンピューティング）、デジタル広告、Prime会員サービス、デバイス（Alexa/Echo/Ring）、Prime Video、Whole Foods（食品小売）、Project Kuiper（衛星通信）を展開。

CEO: Andy Jassy（2021年就任）。従業員数約150万人。米国Eコマース市場シェア約35.7%、クラウド市場シェア約32%で首位。2025年通期売上$716.9B。

🎤 経営陣コメントハイライト
Andy Jassy CEO:
「私たちが今日知っているすべての顧客体験は、AIによって再発明されるだろう。我々はここでリーダーになるために積極的に投資していく。キャパシティを設置するのと同じ速さで収益化している。AWSビジネスの需要シグナルを理解し、そのキャパシティを強力なROIに変える深い経験がある。」
AWS / AI投資:
• AWS +24%成長は過去13四半期で最速。受注残$244B（+40% YoY）で需要が供給を上回る状況が継続
• Trainium2チップを140万個以上展開済み。Trainium3は2026年半ばまでに供給完売見込み
• TrainiumはGPU比で30〜40%優れた価格性能比を実現。Gravitonと合計で年率$10B以上の収益
• AmazonがOpenAIに$50B出資、AWSがFrontierプラットフォームの独占クラウドパートナーに
カンファレンスコール重要ポイント:
• 2026年CapEx $200B（主にAWSのAIインフラ）— アナリスト予想$147Bを大幅超過
• 広告事業は$21.3B（+22% YoY）で急成長、Prime Video広告も寄与
• 食品配送を2026年にさらに多くのコミュニティに拡大、Whole Foodsを100店舗以上新規出店計画
• Project Kuiper: 2026年に20回以上、2027年に30回以上の打ち上げを計画
⚡ リスク・注目ポイント / カタリスト

📈 ポジティブ・カタリスト

  • AWS +24%成長、受注残$244B（+40% YoY）で成長加速
  • Trainium3/4のカスタムチップ戦略でAIインフラの内製化・コスト優位性
  • 広告事業が年間$85B+ペースに急成長（第3の柱）
  • 粗利益率が初の50%突破、利益構造の質的改善
  • 営業CF $139.5B（+20% YoY）と本業の稼ぐ力は堅調
  • アナリスト93%がBuy/Strong Buy、目標株価$290（+36%上方）
  • OpenAIへの$50B出資でAI競争力を補強
  • Prime会員2.4億人超のエコシステム

📉 リスク要因

  • 2026年CapEx $200B計画がFCFを大幅に圧迫。投資回収の不透明感
  • FY25 FCFが$11.2B（前年比-70%）と急激に悪化
  • EPSがコンセンサスを僅差でミス（$1.95 vs $1.97）
  • Azure（+39%）がAWS（+24%）の約2倍の成長率で追い上げ中
  • FTC独禁法訴訟・EU DMA規制の継続的リスク
  • TikTok Shop等の新興Eコマースプレーヤーの台頭
  • International営業利益率がYoYで低下（3.0%→2.1%）
  • Bezosの大型売却（2025年に$6.8B）が継続
