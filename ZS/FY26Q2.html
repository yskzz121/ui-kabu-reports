🎙️ 経営陣コメントハイライト
Jay Chaudhry CEO（AI・ゼロトラスト戦略）
「AIエージェントの時代が到来しており、数十億のAIエージェントが登場する。当社のZero Trust Exchangeは、ユーザー、アプリケーション、AIエージェント間のダイレクトな1対1通信を実現し、ラテラルムーブメントを制限して攻撃面を縮小する設計になっている。」
AI Security の急成長
「2025年暦年で約1兆のAIトランザクションを処理。AI Protectを新たに導入し、AIアセットの発見・管理、アクセスポリシーの適用、プロンプトインジェクション攻撃の防御を実現。AI Security ARRは年度目標$400Mを3四半期前倒しで達成し、年度末$500M超を見込む。」
Zero Trust Everywhere のアップセル効果
「顧客がZero Trust Everywhereに移行すると、ARRは2〜3倍に拡大する。新規ビジネスの25%がメーター制課金で、AIエージェント対応により今後さらに成長が期待される。」
Remo Canessa CFO（財務見通し）
「FY2026通期ガイダンスを引き上げ。ARR $3.73B-$3.75B（24%成長）、Non-GAAP EPS $3.99-$4.02。RPOは前年比31%増の$6.1Bで、将来の収益パイプラインは極めて強固。」
⚠️ リスク・注目ポイント / カタリスト

リスク要因

  • 🔴 オーガニックNet New ARRが市場期待をやや下回り、Red Canary買収による底上げ懸念
  • 🔴 株価が52週高値から54%下落、市場のセンチメント悪化
  • 🔴 GAAP赤字が継続（SBC負担が大きい）、GAAP EPS -$0.21
  • 🔴 Palo Alto Networks・CrowdStrikeとのプラットフォーム競争が激化
  • 🔴 マクロ経済の不確実性とIT支出の選別化
  • 🔴 インサイダーによる株式売却が継続（過去90日で$4.5M相当売却）

カタリスト

  • 🟢 AI Security ARRが80%超成長、FY2026目標を3Q前倒し達成
  • 🟢 RPO +31%で将来収益の可視性が非常に高い
  • 🟢 Zero Trust Everywhereへの移行でARR 2-3x拡大の潜在力
  • 🟢 メーター制課金導入でAIエージェント時代の収益多角化
  • 🟢 サイバーセキュリティ市場は$270B規模、年12%成長（ソフトウェア最速）
  • 🟢 Rule of 40が48.2%と良好、成長と利益のバランスが取れている
